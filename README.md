# WMess

**WMess** — командное рабочее пространство, объединяющее в одном приложении всё, что нужно команде для работы: команды и проекты с гибкими правами доступа, совместную библиотеку документов, таблиц и досок с редактированием в реальном времени, хранилище файлов, чаты с тредами, голосовыми и созвонами, трекер задач (список и Kanban) и общий календарь. По духу — смесь Notion, Mattermost и трекера задач.
